src/lib/models/lang-string.ts

getLangStringValue
getLangStringValue(langString, lang, fallbackLangs)

Extract from a LangString the label in a corresponding lang. If not found, it will try extract the label of one of the fallback languages. If even there nothing is found, it will fallback to the first label found. It is to be noted that even if the label is a string[] with multiple values, only the 0th element will be taken.

Parameters :
Name Optional Description
langString No

The lang string from which we want to extract a label
lang No

The preferred lang to get the label for
fallbackLangs No

ordered list of fallback languages
Returns : string | undefined
